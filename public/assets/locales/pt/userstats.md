## Estatísticas de Usuários e Builds Populares

Esta página agrega dados de usuários do NMS Optimizer para revelar as tecnologias mais frequentemente supercarregadas em diferentes plataformas. Quer você esteja planejando seu próximo build ou procurando inspiração, estas estatísticas apresentam preferências e padrões reais da comunidade.

## Entendendo os Dados

Os gráficos abaixo exibem estatísticas de usuários agregadas dos últimos **28 dias**, mostrando quais tecnologias os jogadores otimizam com mais frequência para **slots supercarregados**. Os dados representam milhares de sessões de otimização em três tipos de plataforma principais:

- **Tecnologias de Espaçonave** - Incluindo variantes de espaçonave padrão, Sentinela e Solar
- **Tecnologias Multi-Ferramenta** - Cobrindo multi-ferramentas padrão, Sentinela, Atlântica e Bastões
- **Tecnologias de Corveta** - Os navios corsários especializados introduzidos na atualização **Omega**

## O Que Significam Esses Gráficos

Cada gráfico mostra a popularidade relativa de diferentes tecnologias quando os usuários escolhem supercarregá-las. Uma fatia maior significa que essa tecnologia foi selecionada para supercarregamento com mais frequência pela comunidade. Isso reflete:

1. **Efetividade Meta** - Tecnologias que fornecem os maiores aumentos de desempenho tendem a se agrupar no topo
2. **Preferências dos Jogadores** - Jogadores focados em combate priorizam tecnologias diferentes daqueles focados em exploração
3. **Oportunidades de Sinergia** - Algumas tecnologias funcionam particularmente bem com supercarregamento, tornando-as escolhas naturais

### Lendo os Gráficos

- Tecnologias representando menos de **2% do total de seleções** são agrupadas na categoria "Outro" para manter os gráficos legíveis
- Percentuais aparecem em cada fatia quando representam pelo menos 6% do total
- As cores das tecnologias correspondem àquelas usadas em todo o otimizador para referência fácil

## Por Que Isso É Importante

Entender os padrões da comunidade pode ajudar a informar suas próprias decisões de build:

- **Validação** - Se uma tecnologia que você está considerando é popular, provavelmente é por uma boa razão
- **Descoberta** - Encontre tecnologias que você pode não ter considerado vendo o que outros priorizam
- **Mudanças Meta** - Observe como os dados mudam conforme o jogo evolui
- **Variedade de Builds** - Use isso como ponto de partida e personalize com base no seu estilo de jogo

## Primeiros Passos

Depois de revisar as opções populares aqui, vá para o [otimizador](/pt) para criar seu próprio build otimizado. Você pode:

1. Selecionar sua plataforma (Espaçonave, Multi-ferramenta, Corveta, Traje ou Exocraft)
2. Configurar seus **slots supercarregados** e **células ativas**
3. Executar o solucionador para encontrar seu layout perfeito

Para instruções detalhadas sobre como usar o otimizador, visite a [página de instruções](/pt/instructions).

## Coleta de Dados

Estes dados são coletados anonimamente a partir de solicitações de otimização e não incluem nenhuma informação de identificação pessoal. Isso ajuda a comunidade do NMS Optimizer a entender quais tecnologias entregam o maior valor e informa melhorias futuras nos algoritmos do solucionador.
