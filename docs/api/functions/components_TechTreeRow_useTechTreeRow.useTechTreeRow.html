      Remarks

      This coordinator hook acts as a facade, aggregating data from multiple stores and delegating specialized tasks to sub-hooks. It handles:

      1. Data aggregation from useGridStore and useTechStore.
      2. Module selection state via useTechModuleManagement.
      3. Optimization and reset lifecycle via useTechOptimization.
      4. Derived logic for localization and resolution-aware image path generation.

      See

      • import('./TechTreeRow').TechTreeRow for the consuming component.
      • useTechModuleManagement for module selection logic.
      • useTechOptimization for solver orchestration.
      • useGridStore for grid state.
      • useTechStore for technology metadata.
