      Custom hook for deriving the control state of a specific grid row.

      Parameters

      • rowIndex: number

        The zero-based index of the row. Must be within grid bounds.

      Returns { isFirstInactiveRow: boolean; isLastActiveRow: boolean }

      Derived boolean flags: isFirstInactiveRow and isLastActiveRow.

      Remarks

      It uses specialized selectors from import('../../store/grid/gridStore').GridStore to determine if a row is the "frontier" for expansion (first inactive row) or the "frontier" for contraction (last active row within the threshold).

      This encapsulates the positioning logic for row activation/deactivation buttons rendered by import('./GridControlButtons').GridControlButtons.
